LISTAGEM E BUSCA

Ao acessar a página de APIs, uma lista com as APIs cadastradas é exibida. Os cards de API exibem a versão da API, além de ícones de ações e acesso a configurações específicas, sendo eles:

  • ícone + para criar nova versão da API
  • ícone de seta para baixo para Exportar
  • ícone de gráfico para Estatística
  • ícone de foguete para Deploy
  • ícone de monitor para Logs
  • ícone de lápis para Editar
  • ícone de lixeira para Excluir
